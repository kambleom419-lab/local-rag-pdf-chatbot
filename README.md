# Local RAG PDF Chatbot

Offline AI chatbot that answers questions from PDFs using:

- LangChain
- Ollama
- ChromaDB
- Phi3-mini

## Features

- Runs fully offline
- Uses vector database search
- Retrieval Augmented Generation (RAG)
- Terminal chatbot interface

## Tech Stack

Python  
LangChain  
Ollama  
ChromaDB

## Run Project

Activate environment

    venv\Scripts\activate

Run chatbot

    python pdf_chatbot.py
